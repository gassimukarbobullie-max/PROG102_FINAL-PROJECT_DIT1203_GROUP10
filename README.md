# 🎓 Student Academic Portal – Limkokwing Sierra Leone

## 📘 Project Overview
The **Student Academic Portal** is a GUI-based Python application developed for the **Principles of Structured Programming (PROG103)** final project at Limkokwing University Sierra Leone.  
It provides a digital solution for managing student records, courses, and results, aligned with **SDG 4 – Quality Education**.

---

## 🎯 Objectives
- Apply structured programming principles in a GUI environment.
- Design modular programs using functions and logical flow.
- Combine GUI components with backend program logic.
- Manage academic data using SQLite3 database.
- Publish and manage the project using GitHub.
- Deliver a solution aligned with Sierra Leone’s educational needs.

---

## ⚙️ Features
- 🔑 **Login Page** – Secure access with demo credentials (`admin / admin123`).  
- 📊 **Dashboard** – Displays live statistics (students, enrolments, courses, GPA).  
- 📝 **Student Registration** – Add or update student records with validation.  
- 📂 **Student Records** – Search and filter student data.  
- 📚 **Course Management** – Add and manage courses.  
- 🎓 **Result Processing** – Automate grade calculation and GPA assignment.  
- 📑 **Reports & Analytics** – Generate academic reports and performance insights.  

---

## 🖥️ Technologies Used
- **Python 3.x**
- **SQLite3** (database)
- **customtkinter** (modern GUI library)
- **PIL (Pillow)** (image handling)
- **CTkMessagebox** (message dialogs)
